# Eyeson demo app

Demo app for Eyeson Api, which allows you to create and join a meeting room, invite participants, use chat during a meeting.

## Main libraries

1. Eyeson Api
2. ViewModel
3. LiveData
4. Coroutines
5. Hilt
6. Retrofit
